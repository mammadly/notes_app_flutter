📱 Simple Notes App – Flutter

A minimal and user-friendly mobile Notes application built with Flutter.
This app allows users to create, edit, and delete notes in a clean and intuitive interface.

🚀 Features
	•	Create new notes
	•	Edit existing notes
	•	Delete notes
	•	Simple and responsive UI
	•	Lightweight and fast performance

🛠 Built With
	•	Flutter
	•	Dart

This project was created as a learning project to improve my Flutter development skills and understand state management and UI building in mobile applications.
